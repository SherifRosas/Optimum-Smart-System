## 🛠️ **TROUBLESHOOTING**

### **If Frontend Won't Start:**
```bash
cd optimum-frontend
npm install
npm start
```

### **If Backend Won't Start:**
```bash
cd optimum-backend
venv\Scripts\activate
python manage.py runserver
```

### **If Port 3000 is Busy:**
- React will ask to use another port (like 3001)
- Just say "Yes" and use the new port
